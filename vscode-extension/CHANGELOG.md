# Changelog

## [1.0.0] - 2026-04-25

### Added
- Generate QR codes from the Command Palette (`Ctrl+Shift+P` → "QRGen: Generate QR Code")
- Automatic CLI dependency detection and install flow
- Styled webview preview panel with QR image display
- Download QR code to custom location
- Copy file path to clipboard
- Open containing folder
- Cross-platform support (Windows, macOS, Linux)
